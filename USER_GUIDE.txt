# 白名单点击 v1.0 用户使用说明

## 一、软件介绍

**白名单点击** 是一款自动化辅助工具，用于根据关键词打开指定页面，识别目标模块，并只点击白名单中的内容。

软件只会点击你设置在白名单里的商家或内容；如果内容在黑名单里，即使也出现在白名单中，也不会点击。

---

## 二、软件启动

收到软件压缩包后，先解压整个文件夹。

解压后打开：

```text
WhitelistClick.exe
```

或者双击：

```text
run_app.bat
```

不要只单独复制 `WhitelistClick.exe`，需要保留整个软件文件夹一起使用。

---

## 三、准备关键词表

关键词表用于填写需要搜索的词。

文件名一般是：

```text
keywords.csv
```

也可以使用：

```text
keywords.xlsx
```

表格格式：

```text
keyword
石家庄亲子鉴定
中山亲子鉴定多少钱一次
北京亲子鉴定中心
```

第一行保留 `keyword`，下面一行一个关键词。

---

## 四、准备白名单表

白名单表用于填写允许点击的商家名称、品牌词或简称。

文件名一般是：

```text
whitelist.csv
```

也可以使用：

```text
whitelist.xlsx
```

推荐格式：

```text
title,url,match_type
石家庄右鉴,,fuzzy
中云联检,,fuzzy
中科生命,,fuzzy
航证基因,,fuzzy
```

说明：

```text
title：白名单名称，可以填完整店名，也可以填简称
url：可以留空
match_type：建议填写 fuzzy
```

如果一个商家有多个名称，可以用 `|` 分开：

```text
石家庄右鉴|右鉴亲子鉴定|右鉴,,fuzzy
```

---

## 五、准备黑名单表

黑名单表用于填写禁止点击的内容。

文件名一般是：

```text
blacklist.csv
```

也可以使用：

```text
blacklist.xlsx
```

格式：

```text
title,url,match_type
广告,,fuzzy
推广,,fuzzy
不想点击的店名,,fuzzy
```

命中黑名单的内容不会被点击。

---

## 六、基础配置

打开软件后，进入 **基础配置** 页面。

常用配置如下：

```text
搜索入口：
https://m.baidu.com/

关键词文件：
选择 keywords.csv 或 keywords.xlsx

白名单文件：
选择 whitelist.csv 或 whitelist.xlsx

黑名单文件：
选择 blacklist.csv 或 blacklist.xlsx
```

修改完成后，点击：

```text
保存配置
```

---

## 七、目标模块配置

进入 **目标模块** 页面。

常用配置：

```text
目标来源标签文字：
百度本地生活

进入页面按钮文字：
查看更多|更多商家|查看全部|全部商家

模块商家 DIV 规则：
div
```

软件会先识别目标模块，进入模块后再判断白名单内容。

没有找到目标模块时，不会点击白名单。

---

## 八、地区设置

进入 **搜索环境/地区** 页面。

如果需要模拟某个城市，填写：

```text
模拟城市：
中山

纬度：
22.5159

经度：
113.3926

时区：
Asia/Shanghai

语言：
zh-CN
```

修改城市后，建议点击：

```text
重置任务进度
```

这样可以避免旧城市缓存影响结果。

---

## 九、动态 IP 设置

如果不使用代理，进入 **动态 IP** 页面，将代理模式设置为：

```text
none
```

如果使用动态 IP，选择：

```text
api
```

然后把代理平台生成的 API 链接填写到：

```text
代理 API URL
```

如果代理需要账号密码，也需要填写：

```text
代理用户名
代理密码
```

如果是白名单验证模式，请先在代理平台把当前电脑的公网 IP 加入白名单。

---

## 十、省流量设置

进入 **省流量** 页面。

推荐选择：

```text
标准省流量
```

标准省流量会减少图片、视频、字体等资源加载，节省动态 IP 流量。

如果发现页面加载不完整，可以改成：

```text
关闭省流量
```

---

## 十一、开始运行

配置完成后，点击右下角：

```text
开始运行
```

软件会自动执行：

```text
读取关键词
↓
打开搜索页面
↓
输入关键词并搜索
↓
识别目标模块
↓
点击查看更多进入模块
↓
识别模块内商家
↓
判断白名单和黑名单
↓
点击白名单内容
↓
停留指定时间
↓
返回继续点击下一个白名单内容
```

---

## 十二、暂停、停止和重置

### 暂停

点击：

```text
暂停
```

任务会临时暂停。

再次点击继续后，会从当前位置继续运行。

### 停止

点击：

```text
停止
```

任务会停止，并保存当前进度。

下次点击“开始运行”时，会从上次进度继续。

### 重置任务进度

如果想从第一个关键词重新开始，点击：

```text
重置任务进度
```

---

## 十三、运行日志

进入 **运行日志** 页面，可以查看当前运行状态。

重点关注：

```text
实时点击白名单数量
```

这个数字会在点击成功后自动增加。

日志里也会显示：

```text
当前关键词
是否找到目标模块
模块内商家数量
白名单命中情况
黑名单拦截情况
点击成功数量
代理状态
异常原因
```

如果软件提供了更新功能，可以点击：

```text
检查更新
```

## 十五、常见问题

### 1. 软件没有点击内容

请检查：

```text
白名单是否填写正确
黑名单是否误拦截
是否找到了目标模块
目标模块文字是否正确
```

### 2. 页面显示了商家，但没有点击

请检查白名单名称是否和页面里的商家名称接近。

例如页面显示：

```text
中云联检基因检测鉴定...
```

白名单可以填写：

```text
中云联检
```

不要只填写过长的完整名称。

### 3. 修改城市后还是旧城市

请操作：

```text
停止任务
↓
重置任务进度
↓
重新开始运行
```

同时确认动态 IP 的地区和模拟城市一致。

### 4. 出现验证码或安全验证

软件不会自动处理验证码。

建议降低运行频率：

```text
并发线程数：1
每个关键词搜索次数：1
搜索后等待：6-8 秒
省流量：标准省流量
```

### 5. 表格读取不到

建议使用 `.xlsx` 格式。

表头建议保持：

```text
title,url,match_type
```

关键词表表头建议保持：

```text
keyword
```

普通用户建议这样设置：

```text
并发线程数：1
每个关键词搜索次数：1
隐藏浏览器：开启
省流量：标准省流量
目标来源标签文字：百度本地生活
进入页面按钮文字：查看更多|更多商家|查看全部|全部商家
模块商家 DIV 规则：div
点击后停留时间：5-10 秒
```

---

## 十七、注意事项

本软件仅用于自有页面、自有商家、排名检测、页面测试等合规用途。

请勿用于恶意点击、刷量、绕过验证码、消耗他人广告预算或其他违规用途。
